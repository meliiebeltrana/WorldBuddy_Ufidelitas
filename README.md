# World Buddy

## Proyecto académico – Universidad Fidélitas

Este repositorio contiene el desarrollo del proyecto **World Buddy**, realizado como parte del curso **Desarrollo de Aplicaciones Web y Patrones** de la **Universidad Fidélitas**.

El proyecto consiste en el diseño y desarrollo de una aplicación web que permite consultar la **hora mundial y el clima de diferentes ciudades del mundo**, facilitando la coordinación entre usuarios ubicados en distintas zonas horarias.

## Descripción del proyecto

World Buddy es una aplicación web que permite a los usuarios consultar información relevante sobre diferentes ciudades del mundo, como la **hora actual y las condiciones climáticas**.

El objetivo del sistema es facilitar la coordinación internacional entre personas que trabajan o interactúan en distintos países, proporcionando información clara y actualizada sobre las zonas horarias y el clima.

---

## Funcionalidades principales

- Consulta de hora mundial por ciudad
- Visualización del clima actual
- Búsqueda de ciudades
- Guardado de ciudades favoritas
- Interfaz amigable para el usuario

## Arquitectura del sistema

El sistema sigue el patrón de arquitectura **Modelo-Vista-Controlador (MVC)**, que permite separar la lógica de negocio, la presentación y el acceso a datos.

Las capas del sistema son:

- **Modelo:** Representa las entidades y la estructura de datos.
- **Vista:** Interfaces HTML que interactúan con el usuario.
- **Controlador:** Gestiona las solicitudes del usuario y coordina la lógica del sistema.
